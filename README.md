ValuEdge Pro Frontend is a professional, finance-focused React + TypeScript dashboard that enables users to run end-to-end company valuations through a clean, analyst-style interface. Built with Vite and Tailwind CSS, it integrates with a live FastAPI backend to deliver real-time DCF outputs, sensitivity matrices, comparable company analysis, and downloadable valuation reports. The application is deployed on Vercel and designed for clarity, speed, and realism in financial modeling workflows.
